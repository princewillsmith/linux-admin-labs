# File Management

## Show current directory

```bash
pwd
```

## List files

```bash
ls
ls -l
ls -la
```

## Create file

```bash
touch test.txt
```

## Create directory

```bash
mkdir projects
```

## Copy

```bash
cp file1 file2
```

## Move

```bash
mv old.txt new.txt
```

## Delete

```bash
rm file.txt
rm -r folder
```

## Find files

```bash
find /home -name "*.txt"
```
